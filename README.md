# Salvida Releases

Öffentliche Builds und Installer für **Salvida** — das PVS-Abrechnungssystem.

Der Quellcode befindet sich im privaten Repository [philglill/salvida](https://github.com/philglill/salvida). Dieses Repo dient ausschließlich der Verteilung der fertigen Builds.

## Download

**Aktuellster Build (immer up-to-date):**

- Windows: [Salvida-Setup.exe](https://github.com/philglill/salvida-releases/releases/latest/download/Salvida-Setup.exe)
- macOS: [Salvida-Mac.zip](https://github.com/philglill/salvida-releases/releases/latest/download/Salvida-Mac.zip)

## Installation

### Windows
1. `Salvida-Setup.exe` herunterladen und ausführen
2. Bei der SmartScreen-Warnung: "Weitere Informationen" → "Trotzdem ausführen"
3. Setup-Wizard durchklicken
4. Salvida wird automatisch im Startmenü eingetragen

### macOS
1. `Salvida-Mac.zip` herunterladen und entpacken
2. `Salvida.app` in den Programme-Ordner ziehen
3. Beim ersten Start: Rechtsklick → Öffnen (wegen Gatekeeper)
